El método includes() determina si una matriz incluye un determinado elemento, devuelve true o false según corresponda.

SINTAXIS: arr.includes(searchElement[, fromIndex])

Donde 'arr' sera el nombre del array, 'searchElement' sera el elemento a buscar y 'fromIndex' la posiscion desde donde se empezara a busar el elemnto, el valor por defecto es 0.
Valor de retorno: Un Boolean que es true si el valor solicitado se encuentra dentro de la matriz (o la parte de la matriz indicada por el índice 'fromIndex', si se especifica).
fromIndex: Si fromIndex es mayor o igual que la longitud de la matriz, se devuelve false. No se buscará en la matriz.
Si fromIndex es negativo, el índice calculado se calcula para usarse como una posición en la matriz en la cual comenzar a buscar searchElement. Si el índice calculado es menor que 0, se buscará la matriz completa.
